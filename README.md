# Инструкция по установке и запуску проекта Tattoo Studio

## Структура проекта

Проект состоит из фронтенда (React) и бэкенда (PHP с MySQL).

## Страницы сайта

1. **Главная** - приветственная страница с информацией о студии
2. **Мастера** - каталог мастеров с портфолио
3. **Мгновенное бронирование** - выбор и бронирование готовых эскизов
4. **Курсы** - продажа обучающих курсов от мастеров
5. **Мерч** - магазин фирменной продукции
6. **Блог** - статьи о татуировках
7. **Контакты** - информация для связи

## Админ-панель

Расположена в `/backend/admin/` и позволяет:
- Управлять мастерами (добавлять, редактировать, удалять)
- Добавлять фото работ мастеров
- Управлять эскизами для бронирования
- Добавлять курсы и мерч
- Вести блог
- Просматривать заявки на бронирование

## Установка

### База данных
1. Создайте БД: `CREATE DATABASE tattoo_studio;`
2. Импортируйте схему из `database/schema.sql`
3. Создайте админа: 
```sql
INSERT INTO admins (username, password_hash) VALUES (admin, y02IXUNpkjO0rOQ5byMi.Ye4oKoEa3Ro9llC/.og/at2.uheWG/igi);
```
(пароль по умолчанию: `password`)

### Бэкенд
Отредактируйте `backend/config.php`:
- Настройте подключение к БД
- Укажите данные ЮMoney для приёма платежей

### Фронтенд
```bash
cd frontend
npm install
npm run dev
```

## Интеграция с ЮMoney

В `backend/config.php` укажите:
- `YUMONEY_SHOP_ID` - ID вашего магазина
- `YUMONEY_SECRET_KEY` - секретный ключ

API для создания платежа: `POST /backend/api/shop.php?action=create_payment`

## Заполнение контентом

Все данные добавляются через админ-панель. Для каждого раздела есть шаблоны с тестовыми данными, которые нужно заменить на реальные.
